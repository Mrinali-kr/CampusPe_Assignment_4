# Chat UI – CampusPe Assignment
Description

This project is a responsive chat user interface built using HTML, CSS, JavaScript, jQuery, and Bootstrap 5.
It simulates a chat experience with user and AI message bubbles, typing animation, and interactive UI components.

Features

* User and AI message display
* Typing indicator and mock AI responses
* Dark/Light theme toggle
* Auto-resizing input box
* Clickable suggestion cards
* Responsive layout

How to Run

1. Open the project folder
2. Open `index.html` in any web browser

No additional setup required.
